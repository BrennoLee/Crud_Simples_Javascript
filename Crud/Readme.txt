CRUD Simples
Um exemplo básico de CRUD (Criar, Ler, Atualizar e Deletar) sem banco de dados, utilizando HTML, CSS e JavaScript.

Visão Geral
Este projeto demonstra um CRUD básico que permite adicionar, visualizar, editar e excluir itens. 
A aplicação é feita apenas com HTML, CSS e JavaScript, sem o uso de banco de dados.

Funcionalidades
Criar: Adicione novos itens à lista.
Ler: Visualize a lista de itens.
Atualizar: Edite itens existentes.
Deletar: Remova itens da lista.

Tecnologias
HTML: Estrutura do conteúdo.
CSS: Estilização e layout.
JavaScript: Lógica de CRUD e manipulação do DOM.

Como Usar
Clone o repositório ou baixe os arquivos.
Abra index.html em um navegador web.
Estrutura do Projeto
index.html: Página principal.
style.css: Arquivo de estilos.
script.js: Lógica de CRUD.

Exemplos de Uso
Para adicionar um item, preencha o formulário e clique em "Enviar".
Para editar um item, clique no botão "Alterar" ao lado do item desejado.
Para excluir um item, clique no botão "Excluir" ao lado do item desejado.
